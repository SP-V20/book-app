# Book App

## Description
This project is a Node.js application designed to maintain book records. 
It allows users to add, retrieve, and delete books from DB.


## Installation
1. Clone the repository:
   ```
   git clone https://github.com/SP-V20/book-app.git
   ```
2. Navigate to the project directory:
   ```
   cd book-app
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage
To start the application, run:
```
node src/index.js
```

## Testing
To run the tests, use:
```
npm test
```
